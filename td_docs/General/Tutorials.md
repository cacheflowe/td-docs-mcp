# Tutorials
##  **TouchDesigner Curriculum**
The curriculum's **[100 Series: TouchDesigner Fundamentals](https://learn.derivative.ca/courses/100-fundamentals/)** course includes 69 individual topics grouped into lessons around a specific learning object in TouchDesigner. Each topic includes a written summary, 5-6 minute video, and example file. Completing this course will teach you the foundational concepts you need to effectively learn from more advanced topics and tutorials.
Following the introductory series, you can move on the the **[200 Series: Intermediate](https://learn.derivative.ca/courses/200-intermediate/)** course for a deeper look at common techniques and workflows.
**Tip:** Create an account and login to track your progress as you complete each lesson.
