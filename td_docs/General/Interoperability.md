# Interoperability
TouchDesigner supports a wide range of devices, protocols and external tools that interface via their respective [Operators](https://docs.derivative.ca/Operator "Operator"), [Palette](https://docs.derivative.ca/Palette "Palette") components, and TouchDesigner Python methods, known collectively as TouchDesigner's Interops.
